# x-engagement-daily

X（Twitter）で指定した18アカウントへの毎日エンゲージメントを自動実行する Cowork スキル。

通知ページでメンション・返信を検出し、各アカウントの最新ツイートにリプライを送信し、関係管理トラッカーファイルを自動更新します。

---

## できること

- Xの通知ページを確認して、対象18アカウントからの返信・メンションを検出
- 各アカウントの最新ツイート（直近24〜48時間）を取得
- 関係度（初回 / 認知 / 継続）に応じたリプ内容を自動生成・送信
- 関係管理トラッカーファイル（Markdown）を自動更新

---

## インストール方法

1. x-engagement-daily.skill ファイルをダウンロード
2. Claudeの Cowork モードを開く
3. スキルファイルをドラッグ＆ドロップしてインストール

---

## 使い方

Coworkで以下のように話しかけるだけで実行されます：

- Xのエンゲージメントタスクを実行して
- 今日のリプ作業やって
- 18アカウントにリプ送って

毎日の定期タスクとして自動実行することもできます（schedule スキルと組み合わせてください）。

---

## 必要な準備

### 1. 関係管理トラッカーファイル

以下のパスに Markdown ファイルを用意してください：

/Users/<ユーザー名>/Documents/Claude/Second Brain/X運用ツール/23_リプ関係管理トラッカー.md

各アカウントのエントリ形式：

### @アカウント名
- 発信テーマ：
- 関係度：初回
- 最終リプ送信：
- 送ったリプ：
- 受信リプ：なし
- メモ：

### 2. Chrome MCP

Claude in Chrome 拡張機能がインストールされ、有効になっていること。

### 3. Xへのログイン

Chromeで X（Twitter）にログイン済みであること。

---

## カスタマイズ方法

### リプ対象アカウントを変更する

Coworkでそのまま話しかけるだけで変更できます：

- 「x-engagement-dailyのリプ対象から @xxx を外して、代わりに @yyy を追加して」

SKILL.md を直接編集する場合は、対象アカウントリストの該当行を追加・削除してください。追加したアカウントはトラッカーファイルにも同じ形式でエントリを追記してください。

---

### 口調・文末パターンを変更する

Coworkで直接伝えるだけで反映されます：

- 「x-engagement-dailyのリプをもっとフランクにして。語尾は だよ だね じゃん 系で」
- 「文末パターンに 〜参考になります！ を追加して」
- 「〜ですよね笑 は使わないようにして」

SKILL.md を直接編集する場合は「トーン・口調ルール」セクションを書き換えてください。

デフォルトの文末パターン：
1. 〜ですね！
2. 〜ですね！w
3. 〜な〜ww
4. 〜わかります笑
5. 〜ですよね笑
6. 〜たしかに〜
7. 〜すごいですね

禁止：「〜ですよ」単体で締めること、同一バッチ内で同じ文末を2回以上使うこと

---

## 関係度の定義

| 関係度 | 条件 | リプのスタンス |
|--------|------|----------------|
| 初回 | 初めてリプする | 丁寧・共感ベース |
| 認知 | 1〜2回やり取りあり | 少しカジュアルに |
| 継続 | 3回以上やり取りあり | 友達感覚、ネタ振りもOK |
| 濃い | 相互フォロー or 頻繁にリプ交換 | ガチ会話、コラボ提案もあり |
