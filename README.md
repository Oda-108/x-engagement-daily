# x-engagement-daily

X（Twitter）で指定した18アカウントへの毎日エンゲージメントを自動実行する Cowork スキル。

通知ページでメンション・返信を検出し、各アカウントの最新ツイートにリプライを送信し、関係管理トラッカーファイルを自動更新します。

---

## できること

- Xの通知ページを確認して、対豦18アカウントからの返信・メンションを検出
- 各アカウントの最新ツイート（直近24〜48時間）を取得
- 関係度（初回 / 認知 / 継続）に応じたリプ内容を自動生成・送信
- 関係管理トラッカーファイル（Markdown）を自動更新

---

## インストール方法

1. `x-engagement-daily.skill` ファイルをダウンロード
2. Claudeの Cowork モードを開く
3. スキルファイルをドラッグ＆ドロップしてインストール

---

## 使い方

Coworkで以下のように話しかけるだけで実行されます：

```
Xのエンゲージメントタスクを実行して
```

```
今日のリプ作業やって
```

```
18アカウントにリプ送って
```

毎日の定期タスクとして自動実行することもできます（`schedule`スキルと組み合わせてください）。

## 必要な準備

### 1. 関係管理トラッカーファイル

以下のパスに Markdown ファイルを用意してください：

```
/Users/<ユーザー名>/Documents/Claude/Second Brain/X運用ツール/23_リプ関係管理トラッカー.md
```

ファイルの構造：

```markdown
## 👥 アカウント別 状態管理

### @アカウント名
- **発信テーマ**：
- **関係度**：🌱 初回
- **最終リプ送信**：
- **送ったリプ**：
- **受信リプ**：なし
- **メモ**：

---

## 📝 リプ送信ログ（時系列）

| 日時 | 送信先 | 元ツイID | 送ったリプ（要約） |
|------|--------|----------|-------------------|

---

## 🔔 受信リプログ

| 日時 | 送信者 | 内容（要約） | 対応状況 |
|------|--------|-------------|------|
```

### 2. Chrome MCP

Xへのアクセスに Chrome MCP が必要です。Cowork の設定でブラウザ操作が有効になっていることを確認してください。

### 3. Xにログイン済みであること

Chromeで `x.com` にログインした状態にしてください。

---

## 口調ルール

- 2行以内・改行で区切る（「/」などの記号は使わない）
- 優しくカジュアルなトーン
- 文末は「ですね！」「ですよね笑」「な～ww」などのバリエーションを使い分け
- **「～ですよ」単体で締めない**
- **同じ実行内で文末パターンを重複しない**

---

## 関係度の定義

| 段階 | 基準 | リプのトーン |
|------|------|-------------|
| 🌱 初回 | 初めてリプを送った | 丁寧かつカジュアル、共感ベース |
| 🌿 認知 | 向こうからリプが返ってきた | より個人的な話題、踏み込んだ感想 |
| 🌳 継続 | 3回以上やり取りあり | 友達感覚、ネタ振りもOK |
| 🔥 濃い | 相互フォロー or 頻繁にリプ交換 | ガチ会話、コラボ提案もあり |
